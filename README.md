# 🚀 Zeyno Bot - WhatsApp Multi Device

Un bot WhatsApp potente, veloce e personalizzato con sistema Owner + Staff.

![Version](https://img.shields.io/badge/Version-1.0.0-blue)
![Node.js](https://img.shields.io/badge/Node.js-18+-green)
![Baileys](https://img.shields.io/badge/Baileys-MD-purple)

---

## ✨ Funzionalità

- **Multi Device** (senza QR ogni volta)
- Sistema **Owner + Staff** completo
- Prefissi multipli: `! . ? / # \`
- Menu bello e organizzato
- Gestione Gruppi (welcome, goodbye, antilink, ecc.)
- Comandi Owner e Staff
- Anti-spam & Anti-crash
- Supporto comandi in gruppo e privato
- Database JSON integrato
- Auto-reload comandi (durante sviluppo)

---

## 📋 Requisiti

- Node.js **v18** o superiore
- Git
- Un numero WhatsApp (non deve essere già collegato a WhatsApp Web)
